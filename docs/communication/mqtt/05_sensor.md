## Sensor Protocols

## Distance Sensor

### /sensor/distance/{robotID}/?

| Property       | Value                                                                                                                             |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| Source         | Simulator                                                                                                                         |
| Destination    | Robot (Physical &#124; Virtual)                                                                                                   |
| Data Type      | String                                                                                                                            |
| Message Format | `?`                                                                                                                               |
| Description    | This will request distance sensor readings from a robot. Typically used by the Simulator or Sandbox.<br><br>**robotID**: Robot ID |

### /sensor/distance/{robotID}

| Property       | Value                                                                                                                                                                                              |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Source         | Simulator                                                                                                                                                                                          |
| Destination    | Robot                                                                                                                                                                                              |
| Data Type      | String                                                                                                                                                                                             |
| Message Format | `{distance}`                                                                                                                                                                                       |
| Description    | Simulator will inform Mixed Reality Environment readings to the robot, as a reply to the topic `/sensor/distance`<br><br>**robotID**: Robot ID<br>**distance**: Distance in centimeters as a float |

### /sensor/distance

| Property       | Value                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Source         | Robot                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Destination    | Simulator                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Data Type      | JSON                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Message Format | `{"id": [robotID], "dist": [distance], "reality": "M"}`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| Description    | Robots can request mixed-reality sensor reading from simulator through this topic. There as an optional parameter, `'reality'` is used to request the reading only on a specified reality. Reply from the simulator will be received through the topic `/sensor/distance/{robotID}`.<br><br>Optionally, physical robots can inform their distance sensor readings to the server using dist parameters.<br><br>**robotID**: Robot ID<br>**dist(optional)**: Distance in centimeters as a float<br>**reality(optional)**: M: mixed, V: virtual, R: real &#124; default: V |

## Color Sensor

### /sensor/color/{robotID}/?

| Property       | Value                                                                                             |
| -------------- | ------------------------------------------------------------------------------------------------- |
| Source         | Simulator                                                                                         |
| Destination    | Robot (Physical &#124; Virtual)                                                                   |
| Data Type      | String                                                                                            |
| Message Format | `?`                                                                                               |
| Description    | This will request color sensor readings from a robot. Typically used by the Simulator or Sandbox. |

### /sensor/color/{robotID}

| Property       | Value                                                                                                                                                                                                                                                                                       |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Source         | Simulator                                                                                                                                                                                                                                                                                   |
| Destination    | Robot                                                                                                                                                                                                                                                                                       |
| Data Type      | String                                                                                                                                                                                                                                                                                      |
| Message Format | `{R} {G} {B} {ambient}`                                                                                                                                                                                                                                                                     |
| Description    | Simulator will inform Mixed Reality Environment readings to the robot. Sensor readings must wait for this reply before deciding the final value.<br><br>**R,G,B**: corresponding color reading &#124; range: [0,255]<br>**ambient**: Ambient light of the environment &#124; range: [0,255] |

### /sensor/color

| Property       | Value                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Source         | Physical Robot                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Destination    | Simulator                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Data Type      | JSON                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Message Format | `{"id":[robotID], "R":[R], "G":[G], "B":[B], "ambient":[ambient], "reality": "M"}`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Description    | Robots can request mixed-reality sensor reading through this topic. There as an optional parameter, `'reality'` is used to request the reading only on a specified reality. Reply from the simulator will be received through the topic `/sensor/distance/{robotID}`.<br><br>Optionally, physical robots can inform their sensor readings to the server using R,G,B, ambient parameters.<br><br>**id**: Robot ID<br>**reality(optional)**: M: mixed, V: virtual, R: real &#124; default: V<br>**R,G,B (optional)**: corresponding color reading &#124; range: [0,255]<br>**ambient (optional)**: Ambient light of the environment &#124; range: [0,255] |

## Proximity Sensor

### /sensor/proximity/{robotID}

| Property       | Value                                                                                                                                                                                                                                                                                                                                                                                                                            |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Source         | Simulator                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Destination    | Robot (Physical, Virtual)                                                                                                                                                                                                                                                                                                                                                                                                        |
| Data Type      | String                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Message Format | `[ vary with the request ]`                                                                                                                                                                                                                                                                                                                                                                                                      |
| Description    | Simulator will update virtual proximity sensor readings to the robot, as the reply to the '/sensor/proximity' topic. This is a full virtual sensor<br><br>The response will be depend of the requested number of angles (it can be decided by the robot)<br><br>Example: for angels = [-90, 0, 90]<br>`5 10 5 #404040 #000000 #FF0000`<br>{dist on -90°} {dist on 0°} {dist on 90°} {color of -90°} {color of 0°} {color of 90°} |

### /sensor/proximity

| Property       | Value                                                                                                                                                                                                                                                                                                                          |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Source         | Robot (Physical, Virtual)                                                                                                                                                                                                                                                                                                      |
| Destination    | Simulator                                                                                                                                                                                                                                                                                                                      |
| Data Type      | JSON                                                                                                                                                                                                                                                                                                                           |
| Message Format | `{"id":0, "angles": [[angles]], "reality":[reality]}`                                                                                                                                                                                                                                                                          |
| Description    | The sensor is entirely virtual and both virtual and physical robots can request the proximity readings from the simulator using this.<br><br>**angles**: relative angles that measures should be taken, as an integer list. Ex: `[-90, 0, 90]`<br>**reality(optional)**: V: virtual(default, and the only possible option far) |
