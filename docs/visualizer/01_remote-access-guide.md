## Remote Access Guide

The Visualizer Tools can be accessed remotely from the below page.

- [https://pera-swarm.ce.pdn.ac.lk/login/](https://pera-swarm.ce.pdn.ac.lk/login/)

You had to provide the valid credentials of the simulation server.

- `host` and `port` of the MQTT broker
- `username` and `password` for the MQTT broker's user accounts
- `channel`: Simulator channel

After filling these details, you can generate a valid link to each available simulator, with valid Access Token. Once you visit the site, the access token is saved on the browser and no need to join through the login page again, unless you need to switch into a different simulator channel. You can also share the generated link with your team members to give them access to the same simulator channel.
